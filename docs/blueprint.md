# **App Name**: SortifyAI

## Core Features:

- Image Upload and Text Query: Allows users to upload waste images or enter text descriptions for classification.
- AI Waste Classification: Classifies waste into categories (paper, plastic, metal, glass, biodegradable, hazardous, textile) using a pre-trained PyTorch model. This model will tool user disposal tips and information for better user experience.
- Disposal Instructions: Provides specific disposal instructions for each waste category (recycle, compost, hazardous handling).
- Reward System: Assigns reward points or coupons to users based on their correct waste classifications.
- User Data Storage: Stores user data, including reward points, using Firebase.
- Mobile-Friendly UI: Provides a responsive and lightweight user interface optimized for mobile devices using React.js.

## Style Guidelines:

- Primary color: Forest green (#388E3C), representing environmental awareness and sustainability.
- Background color: Light grey-green (#E8F5E9), offering a clean and calming backdrop.
- Accent color: Yellow-orange (#FBC02D) to highlight interactive elements and rewards.
- Body and headline font: 'PT Sans', a humanist sans-serif, combines a modern look and a little warmth or personality; suitable for headlines or body text.
- Use simple, intuitive icons representing each waste category.
- Clean and minimalist layout, ensuring ease of navigation and readability.
- Subtle animations to provide feedback on user actions and improve engagement.
